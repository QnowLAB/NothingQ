## Quick Start

Click the emoji targets as fast as you can. Avoid missing targets and bombs. Reach higher levels for speed challenges.

### Key Features
- ❤️ 3 lives to survive
- 🏆 Score and Best Score tracking
- 🪙 Coins collection
- 📈 Level progression
- 🔥 Combo multiplier
- ✨ Particle effects
