# Changelog

All notable changes to NothingQ are documented here.

---

## V3.1

**Released:** July 10, 2026

### Added
- ❤️ Lives system (start with 3 lives)
- Game Over modal showing final score and best score
- Restart button (preserves Best Score across sessions)
- "Nothing is Impossible" slogan
- Lives display in HUD

### Fixed
- Premature life loss on game start
- Target miss detection logic (now only counts after first target)
- Prevented double life penalties on bomb hits

---

## V2.9

First Fun Release

### Features
- Random target spawn mechanics
- Level progression system
- Coins collection
- Combo multiplier
- Best Score tracking
- Particle effects on target hit
- Level Up animations
- 8 emoji targets with different point values
- Vibration feedback on mobile
